GeoPackage Leaflet Plugin

by the National Geospatial-Intelligence Agency

GitHub NPM Demo

The GeoPackage Leaflet Plugin library was developed by the National Geospatial-Intelligence Agency (NGA) in collaboration with BIT Systems.

This library creates layers in leaflet for displaying GeoPackage tile and vector tables.
